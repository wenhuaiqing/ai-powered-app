# ai-powered-app

Mock of Reapit's "One Platform" — a portfolio demo built around Sydney house-price Kaggle data. Showcases multi-agent AI (LangGraph), RAG (NSW regulations), text-to-DuckDB, live web search (Tavily), and a RandomForest valuation model. Frontend mirrors Reapit's purple/teal visual identity.

## Tech stack

- **Backend**: FastAPI 0.111+, Python 3.12, uv. DuckDB for analytics. LangGraph + `openai` SDK (Azure-pointed). scikit-learn for ML. Tavily for web search. Local cosine over parquet for RAG.
- **Frontend**: React 18.3, Vite 5, React Router 7, Recharts, Leaflet, `react-ai-orb`, lucide-react.
- **AI**: Azure OpenAI for dev (chat + embeddings). AWS Bedrock migration is Phase 2.

## How to run

```powershell
# 1) Build data + model (one-time, regenerate after CSV changes)
cd backend
uv sync
uv run python ../scripts/build_db.py
uv run python ../scripts/train_model.py
uv run python ../scripts/build_review_embeddings.py
uv run python ../scripts/build_regulation_corpus.py

# 2) Backend
uv run uvicorn src.main:app --reload --port 8000

# 3) Frontend (separate terminal)
cd frontend
npm install
npm run dev   # :5173
```

Tests: `cd backend && uv run pytest`.

## Conventions

- **No emojis in code or output unless asked.**
- **Australian English** throughout (colour, organise, metre). Dates DD/MM/YYYY where displayed.
- Thin routers — handler ≤20 lines; SQL + business logic in `services/`.
- Every agent emits a Pydantic-typed JSON object (structured outputs). Never free text into graph state.
- LangGraph is the harness — per-node 30s timeout, 90s end-to-end, 1 retry on validation errors.
- The Orb is the primary user surface — streamed SSE events render as agent-trace cards.
